# Bundle Discount Calculator

This project is a Bundle Discount Calculator that helps users calculate discounts when purchasing multiple items together. 

## Features
- Calculate total price after applying bundle discounts.
- User-friendly interface for easy input of items and prices.

## Usage Instructions
1. Clone the repository:
   ```
   git clone https://github.com/taka-nakani/BundleDiscountCalculator.git
   ```
2. Navigate to the project directory:
   ```
   cd BundleDiscountCalculator
   ```
3. Open the application and follow the on-screen instructions to enter item prices and calculate discounts.
